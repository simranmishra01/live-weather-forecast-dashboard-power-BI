# 🌦️ Live Weather Forecast Dashboard — Power BI  

> Real-time multi-city weather monitoring dashboard powered by Weather API integration.

---

## 📌 Overview  

This project presents an **interactive Live Weather Forecast Dashboard** built using **Power BI**, designed to monitor real-time weather conditions across multiple cities.

The dashboard integrates live weather data through an API connection, enabling refresh-driven updates that provide the latest environmental conditions and forecast information. It allows users to compare weather across cities, track air quality levels, and analyze short-term temperature trends in a single interface.

**Key focus areas:**  
Multi-city weather monitoring · 7-day forecast · AQI tracking · Daylight cycle visualization · Environmental metrics · Refresh-enabled updates

## 🚀 Features & Highlights  

### 🔍 Problem Statement  

Monitoring weather across multiple cities typically requires switching between multiple applications or websites. This creates inefficiencies in comparing weather conditions and environmental data.  

This project addresses that challenge by providing a **centralized and interactive weather dashboard** that delivers real-time insights in one place.

---

## 📂 Data Source  

Weather data used in this project is retrieved through a **Weather API**, providing structured real-time weather and environmental data.

### Data Fields Included:

- City Name  
- Temperature  
- Weather Condition (Mist, Sunny, etc.)  
- Humidity  
- Pressure  
- Wind Speed  
- Visibility  
- UV Index  
- Precipitation  
- Air Quality Metrics (PM2.5, CO, NO2, O3, PM10, SO2)  

---

## 🎯 Project Objective  

To build a **dynamic and interactive Power BI dashboard** that allows users to monitor current weather conditions, analyze air quality, and compare multi-city forecasts using refresh-enabled data.

---

## 📊 Dashboard Components  

| Visual | Type | Purpose |
|-------|------|--------|
| Current Weather Panel | KPI Card | Displays temperature and weather condition |
| Environmental Metrics | KPI Cards | Shows humidity, pressure, wind speed, UV index |
| Daylight Cycle | Icon Cards | Displays sunrise and sunset timing |
| Air Quality Index | Donut Chart | Visualizes PM2.5 and air quality metrics |
| Pollutant Indicators | KPI Cards | Shows CO, NO2, O3, PM10, SO2 levels |
| 7-Day Forecast | Line Chart | Displays temperature trends across days |
| Daily Forecast Panel | Cards | Shows day-wise temperature summary |
| City Selector | Buttons/Slicers | Enables multi-city comparison |
